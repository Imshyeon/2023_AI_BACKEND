primitive type : String, Number, Boolean, Null, Undefined 
            -> 데이터 복사할 때 깊은 복사가 되는 타입
        reference type : Object 
            -> 데이터 복사할 때 얕은 복사가 되는 타입(주소 대입)

    
        querySelector : css 스타일 기반 요소를 선택하는 함수 - > 첫번째 요소
                        ex. let BB = document.querySelector("#BB"); => id = BB
                                     document.querySelector(".note"); => class=".note"

        getElementById : id만 선택하는 함수, .note를 모른다!
                        ex. document.getElementById("BB"); => id = BB
