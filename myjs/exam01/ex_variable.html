var 변수명 -> 전역으로 씀 (es6 이전에는 얘를 권장) : 함수라벨 스코프
        let 변수명 -> 지역 ( 현 페이지에만 적용 , es6 이후에는 얘를 권장) : 블럭 라벨 스코프
                      {}나 함수 안에 선언..
        const 변수명 -> 상수 (es6 이후 권장) : 블럭 라벨 스코프
                      
        스코프 scope : 메모리에 변수가 올라가고 내려오는 기준 또는 범위
    
        window -> 전역객체 : 자바스크립트에서 사용하는 변수, 함수 객체 속성 등
                            프로그램에서 제공해주는 전역객체
            - tip : 전역객체는 브라우저가 종료될 때까지 메모리에 소거되지 않는 객체.
            - 전역에서 var로 선언한 변수는 전역객체인 window 속성으로 등록이 된다
            - let, const로 선언한 변수는 전역객체인 wisdow 속성으로 등록되지 않는다.
